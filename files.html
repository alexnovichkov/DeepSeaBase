DeepSea Base

Работа с записями и каналами

DeepSea Base поддерживает работу с файлами следующих типов:

  • Файлы DeepSea (*.dfd + *.raw)
  • Файлы Universal File Format текстового формата (*.uff, *.unv)

Добавление и удаление записей

Добавить записи в базу можно как в текущую вкладку, так и в новую. Можно добавлять:

  • Записи из папки (Ctrl+O)
  • Записи из папки и всех вложенных в нее папок
  • Файлы по одному / по нескольку

Чтобы удалить запись/записи, выделите ее/их в таблице записей и нажмите Del.

Изменение каналов в записях

Удаление каналов

Отметьте галочками каналы, которые вы хотите удалить. Затем нажмите кнопку .

Выпадающее меню в этой кнопке позволяет удалять каналы сразу из нескольких файлов. Для этого в левой таблице файлов выделите требуемые файлы, отметьте галочками ненужные каналы и выберите пункт меню "Удалить каналы в нескольких файлах".

Перемещение и копирование каналов

Доступно в выпадающем меню "Операции...".
