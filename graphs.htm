Построение графиков

Добавление и удаление графиков

  1. Выбрать файл в левой таблице файлов.
  2. Отметить нужные каналы (все каналы файла) в правой таблице каналов.
  3. Если выделить несколько файлов, и отмечать каналы с зажатой клавишей Ctrl, то будут построены все каналы с этим номером в выделенных файлах.
  4. Удалить график можно, убрав галочку напротив канала в правой таблице или выбрав канал в легенде графика правой кнопкой мыши.

Замечания

  1. Каналы с временными реализациями будут построены не целиком. По умолчанию берутся первые 200 отсчетов.

Управление графиками

Внутри графика

  • Правая кнопка мыши: двигать график по плоскости
  • Правая кнопка мыши + Ctrl: двигать график по плоскости относительно правой оси
  • Колесико мыши: увеличить/уменьшить масштаб
  • Колесико мыши + Ctrl: увеличить/уменьшить масштаб по горизонтали
  • Колесико мыши + Shift: увеличить/уменьшить масштаб по вертикали
  • Левая кнопка мыши: масштабирование по выделению. Чтобы прервать начатое выделение, нажмите кнопку ESC.
  • Backspace или двойной щелчок: вернуть масштаб к исходному
  • Ctrl + Backspace: вернуть масштаб к исходному по горизонтали
  • Shift + Backspace: вернуть масштаб к исходному по вертикали

На осях

  • Левая кнопка мыши: увеличить/уменьшить масштаб на оси
  • Контекстное меню: изменить формат оси (линейный/логарифмический)

Форматирование графиков

Можно изменять оформление построенных графиков. Для этого необходимо левой кнопкой мыши выбрать кривую в легенде графика. Откроется диалоговое окно "Настройки графика", где можно:

  • изменить название кривой в легенде
  • изменить цвет, толщину и тип линии
  • переместить кривую на дополнительную вертикальную ось

Нажатием клавиши H можно показать/спрятать подписи осей.

Подписи дискрет

  • Чтобы добавить подпись к выделенной точке, необходимо нажать ПРОБЕЛ. При этом значение по оси X, отображаемое по умолчанию при выделении точек, сохранится на графике.
  • Чтобы изменить формат подписи, необходимо выделить подпись левой кнопкой мыши (при этом подпись будет обведена рамкой) и нажать пробел
  • Форматы подписей (переключаются циклически при нажатии пробела)
    1. значение по оси X (по умолчанию)
    2. значение по оси X + значение по оси Y
    3. значение по оси Y
  • Чтобы удалить подпись, нужно выделить ее и нажать клавишу DELETE.
