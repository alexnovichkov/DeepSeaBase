v.1.6.7

  • Добавлен диалог изменения масштаба оси по двойному щелчку на оси.
  • Добавлены кнопки включения/выключения автомасштабирования по всем трем осям.
  • Добавлена кнопка подгонки масштаба всего графика.
  • Добавлена возможность строить графики сразу на правой оси Y.
  • Добавлена заливка площади внутри курсоров дискрет.
  • Добавлены два курсора режекции.
  • Добавлен контроль количества усреднений при расчете спектров.
  • Добавлена децимация в методе "Осциллограф".
  • Добавлено сохранение вырезки временных данных.
  • Добавлено: если построены графики каналов с одинаковыми названиями, то в легенде показывается номер файла в таблице файлов.
  • Добавлено быстрое открытие файлов uff.
  • Добавлена возможность менять формат отображения шкалы Y в контекстном меню оси Y: амплитуды линейные / амплитуды в дБ / действ. часть / мнимая часть / фаза.
  • Добавлен режим Сергея построения графиков: при перемещении по записям автоматически строятся каналы, которые уже были построены ранее.
  • Добавлена кнопка, позволяющая одним нажатием открыть файл для редактирования в текстовом редакторе.
  • Добавлена возможность скрывать с графика кривые, не удаляя их.
  • Полностью переделана легенда графика - теперь выделенная кривая не скачет в самый низ легенды.
  • Исправлена работа с масштабированием графика путем выделения при включенных курсорах.
  • Исправлена ошибка при удалении метки с графика.
  • Исправлено отображение шкалы Х в логарифмическом масштабе.
  • Исправлены курсоры для третьоктав.
  • Исправлено падение программы при чтении поврежденных файлов raw.
  • Исправлено чтение старого и нового названия оси Y для канала, содержащего величины в дБ.
  • Исправлен экспорт в Excel графиков каналов с разным количеством отсчетов.
  • Увеличено количество стандартных цветов графиков с 16 до 32. Цветовая схема изменена на схему Test.Xpress.
  • Оптимизирована отрисовка графиков с большим количеством точек - графики временных данных теперь строятся в 5-10 раз быстрее.
  • Изменен способ зуммирования (изменения масштаба): убраны модификаторы Ctrl, теперь зуммирование колесиком доступно при наведении на ось координат и на сам график.
  • Изменен способ чтения файлов dfd, у которых начало шкалы X не равно нулю.
