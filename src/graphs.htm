Построение графиков

Добавление и удаление кривых

  1. Выбрать файл в левой таблице файлов.
  2. Отметить нужные каналы (все каналы файла) в правой таблице каналов.
  3. Если выделить несколько файлов, и отмечать каналы с зажатой клавишей Ctrl, то будут построены все каналы с этим номером в выделенных файлах.
  4. Удалить кривую можно, убрав галочку напротив канала в правой таблице или выбрав канал в легенде графика правой кнопкой мыши.
  5. Скрыть кривую можно, убрав галочку напротив канала в легенде графика.

Замечания

  1. Каналы с временными реализациями будут построены с фильтрацией данных. Это может влиять на эстетическое восприятие графика.
  2. Можно сразу построить все каналы файла по правой оси Y. Для этого нужно нажатием правой кнопки мыши на файле (файлах) открыть контекстное меню и выбрать пункт "...на правой оси Y".
  3. Специальный режим построения кривых ("Режим Сергея") позволяет быстро просмотреть определенные каналы списка файлов. Включите режим Сергея кнопкой , постройте нужные каналы. Теперь при выделении файла в левой таблице файлов автоматически будут построены каналы с теми же номерами, а старые кривые будут удалены. (Описание режима сложнее, чем его использование).

Управление кривыми

Внутри графика

  • Правая кнопка мыши: двигать график по плоскости
  • Правая кнопка мыши + Ctrl : двигать график по плоскости относительно правой оси
  • Колесико мыши: увеличить/уменьшить масштаб сразу по всем осям
  • Левая кнопка мыши: масштабирование по выделению. Чтобы прервать начатое выделение, нажмите кнопку Esc.
  • Backspace: вернуть масштаб к предыдущему

На осях

Чтобы растянуть график в определенном направлении, левой кнопкой мыши схватите часть оси координат (горизонтальную или одну из вертикальных), и потяните вдоль оси.

  • Растянуть правую границу графика - схватите правую часть горизонтальной оси и потяните влево/вправо.
  • Растянуть левую границу графика - схватите левую часть горизонтальной оси и потяните влево/вправо.
  • Растянуть верхнюю границу графика - схватите верхнюю часть вертикальной оси (левой или правой) и потяните вверх/вниз.
  • Растянуть нижнюю границу графика - схватите нижнюю часть вертикальной оси (левой или правой) и потяните вверх/вниз.

Чтобы увеличить/уменьшить масштаб вдоль определенной оси, наведите курсор мыши на эту ось и колесиком мыши измените масштаб.

Чтобы задать точный диапазон по оси X, двойным щелчком по оси X откройте диалоговое окно "Установка шкалы" и введите минимальное и максимальное значения. Опция "Подогнать масштаб по вертикальным осям" позволяет вписать график по оси Y (не работает, как задумано).

В контекстном меню оси X можно изменить формат оси (линейный/логарифмический).

В легенде

Управление кривыми в легенде осуществляется мышью:

  • Левая кнопка мыши - изменить форматирование кривой (см. ниже).
  • Средняя кнопка мыши - переместить кривую на правую / левую ось Y.
  • Правая кнопка мыши - удалить кривую.

Текущая выделенная кривая отображается в легенде подчеркиванием названия.

Автомасштабирование кривых

По умолчанию добавляемые на экран кривые автоматически масштабируются по всем осям. Ести это вас не устраивает, автомасштабирование можно отключить кнопками , , .

Кнопка позволяет автомасштабировать график сразу по всем осям.

Отображение кривых

Формат файла uff позволяет сохранять комплексные данные о спектрах. Поэтому в версию 1.6.7 была добавлена возможность менять формат отображения кривых.

В контекстном меню оси Y (правой или левой) выберите один из форматов оси:

  • Амплитуды линейные
  • Амплитуды в дБ
  • Фазы
  • Действительные
  • Мнимые

Если данные хранятся в комплексной форме, то доступны все пять форматов. В противном случае всё зависит от формата данных (это особенно касается файлов DFD). Например, если в файле сохранены только амплитуды в дБ, то вместо действительных, мнимых значений и фаз будут построены нули.

Для контроля текущего формата отображения рядом с названием оси Y в квадратных скобках указывается формат оси.

Форматирование кривых

Можно изменять оформление построенных кривых. Для этого необходимо левой кнопкой мыши выбрать кривую в легенде графика. Откроется диалоговое окно "Настройки кривой", где можно:

  • изменить название кривой в легенде
  • изменить цвет, толщину и тип линии
  • переместить кривую на дополнительную вертикальную ось

Нажатием клавиши H можно показать/спрятать подписи осей.

Кнопка позволяет включить или отключить отображение текущих координат курсора мыши (в системе координат графика). Значение по оси Y соответствует левой вертикальной оси (даже если кривые построены только по правой оси).

Подписи дискрет

  • Чтобы добавить подпись к выделенной точке, необходимо нажать ПРОБЕЛ. При этом значение по оси X, отображаемое по умолчанию при выделении точек, сохранится на графике.
  • Чтобы изменить формат подписи, необходимо выделить подпись левой кнопкой мыши (при этом подпись будет обведена рамкой) и нажать пробел
  • Форматы подписей (переключаются циклически при нажатии пробела)
    1. значение по оси X (по умолчанию)
    2. значение по оси X + значение по оси Y
    3. значение по оси Y
  • Чтобы удалить подпись, нужно выделить ее и нажать клавишу Del.

Режим изменения данных

Включается кнопкой

Используется, если необходимо мышью сдвинуть дискрету выше или ниже. Щелкните левой кнопкой мыши по точке и, не отпуская кнопку, переместите точку вверх или вниз.

Отображение координат курсора мыши должно быть включено (кнопка ).

Замечания

  • Измененное значение дискреты будет сохранено в файл. Пока что отключить сохранение нельзя.

Курсоры дискрет

Включаются кнопкой

Позволяют выводить в отдельное диалоговое окно значения дискрет для определенного значения по оси X. Всего можно задать четыре курсора:

  • два курсора "нормальные" (условно левый и правый, хотя они могут размещаться как угодно относительно друг друга). Синего цвета, с заливкой внутренней области синим.
  • два курсора режекции, красного цвета, с заливкой внутренней области красным.

Способы задания положения курсоров:

  • Левой кнопкой мыши внутри графика
  • Заданием точного значения по оси X
  • Перетаскиванием курсора левой кнопкой мыши
  • Стрелками "влево" и "вправо", если привязать курсор к точке на кривой. Для привязки щелкните левой кнопкой мыши по любому графику, чтобы выделилась точка на графике. Курсор дискрет будет привязан к значению по оси X этой точки.

Замечание: курсоры режекции можно добавить только в окне дискрет.

В диалоговом окне курсоров также рассчитываются среднеквадратичное значение и энергия в полосе (от левого курсора до правого). Если дискреты попадают в полосу режекции, то дополнительно рассчитывается величина режекции.

Отдельная опция позволяет выводить значения дискрет вдоль линии курсора.

Можно также включить показ гармоник (для положения левого курсора).
