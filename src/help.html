DeepSea Base

Возможности программы

Программа DeepSea Base предназначена для быстрого просмотра графиков сигналов. Она умеет:

  • Открывать файлы следующих типов: Файлы Deepsea (*.dfd), Файлы UFF (текстового формата, *.uff), Файлы D94 (*.d94), Файлы ANA/ANP (*.anp), Файлы WAV (*.wav)
  • Конвертировать файлы следующих типов: Файлы National Instruments (*.tdms, *.tdm), Файлы Matlab, сохраненные из Test.Xpress (*.mat), Файлы D94 (*.d94), Файлы ESO (*.eso)
  • Выполнять расчеты спектров с использованием встроенного движка или с помощью DeepSea
  • Удалять, вставлять, перемещать каналы в записях и между записями
  • Изменять данные: добавлять поправки, менять уровень отдельных дискрет
  • Строить графики каналов разных типов: временных характеристик, спектральных характеристик, долеоктавных спектров, сонограмм
  • Экспортировать графики и данные в Excel, в виде рисунка, печатать на принтере

Программа способна логировать свои действия в файл-лог. Для этого её следует запустить из консоли с параметром -l или --log.

По умолчанию логируются только основные действия пользователя (создание/удаление вкладок, добавление файлов, построение графиков, расчет спектров и т.д.). Для поиска ошибок в программе существует возможность логировать каждую операцию программы. Для этого программу следует запускать с параметром -v. Число после параметра указывает уровень логирования (1 - только основные операции, 2 - основные операции и дополнительные, 3 - все операции программы).
